Lancer l'application en exécutant la classe HelloWorld.java

- La première fenêtre vous permet de vous connecter à l'aide d'un couple Nom/Prénom
- Si le couple est connu de l'application, l'utilisateur est redirigé vers la liste des coffres possédés par ce dernier
- L'utilisateur peut ensuite réaliser différentes actions
- A la cloture de l'application, l'ensemble des données sont stockées dans un fichier xml
